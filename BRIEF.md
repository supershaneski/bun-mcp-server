# HTTP MCP Server

## 1. Core Promise
We are building an HTTP MCP server that should be able to work with actual MCP client.

## 2. Emotional Outcome
We should be able to go over the whole lifecycle of an MCP server and sending proper response when error occurs.

## 3. Overview
* We are using Bun to create and scaffold our HTTP server.
* We aim to separate concerns in code like routing and MCP request handling.

## 4. Scope Boundaries

This is merely a project for learning how MCP server works and is not meant to be exhaustive unless we update its objective later on.
